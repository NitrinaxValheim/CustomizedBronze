# CustomizedBronze

Makes the recipe for Bronze customizable by the user.

* If you want to use the standard alloy from Valheim, set the preset to *Default*. (2 copper + 1 tin equals 1 bronze)
* If you want to use an alloy like the one in World of Warcraft, set the preset to *WoW-like*. (1 copper + 1 tin equals 2 bronze)
* If you want to use a more realistic alloy, set the preset to *Realistic*. (2 copper + 1 tin equals 3 bronze)
* If you want an alloy with a custom mix ratio, set the preset to *Custom* and adjust the values below as you wish.

* ~~Please note: After changing settings, you must log out and log in for the changes to be applied.~~
* New: applied immediately (no re-login required)
